## 🔧 Scripts Disponibles

### generate-image.sh
Script bash para generar imágenes con Nano Banana 2.

**Uso:**
```bash
./generate-image.sh "tu prompt aquí"
./generate-image.sh "prompt" 4              # 4 imágenes
./generate-image.sh "prompt" 1 16:9         # Relación 16:9
./generate-image.sh "prompt" 1 1:1 4K       # Resolución 4K
```
